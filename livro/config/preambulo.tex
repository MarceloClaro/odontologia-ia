% Formatação de títulos conforme NBR 6024:2012 (Numeração Progressiva)
% Hierarquia tipográfica obrigatória: maiúscula, negrito, itálico
% Seção primária   (capítulo):       MAIÚSCULAS + NEGRITO
% Seção secundária (seção):          MAIÚSCULAS + NEGRITO
% Seção terciária  (subseção):       Primeira maiúscula + NEGRITO
% Seção quaternária (subsubseção):   Primeira maiúscula + ITÁLICO (sem negrito)
