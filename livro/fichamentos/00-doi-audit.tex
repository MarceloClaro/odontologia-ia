\section{Recomendações}
%==============================================================================

\begin{enumerate}
  \item \textbf{Crítico (35,5\% dos DOIs):} Os 22 DOIs com status 404
        representam o risco mais grave à credibilidade acadêmica do livro.
        Recomenda-se:
        \begin{itemize}
          \item Verificar manualmente cada referência no PubMed, Scopus
                ou Web of Science;
          \item Substituir o DOI por outro válido ou remover a citação
                se o artigo não for localizado;
          \item Documentar o resultado da verificação em
                \texttt{CORRIGENDUM.md}.
        \end{itemize}
  \item \textbf{Atenção (19,4\% dos DOIs):} Os 12 DOIs com acesso
        restrito (403) existem e são válidos, mas exigem assinatura
        institucional ou pagamento para acesso ao texto completo. O
        DOI como identificador persistente é funcional; recomenda-se
        que o autor verifique se possui acesso institucional para
        confirmar o conteúdo.
  \item \textbf{Satisfatório (45,2\% dos DOIs):} Os 28 DOIs verificados
        com sucesso (200) estão ativos e acessíveis. Nenhuma ação
        requerida.
  \item \underline{Auditoria adicional:} Após a correção dos DOIs com
        404, repetir este procedimento de verificação e atualizar este
        relatório.
\end{enumerate}

\section{Metodologia}
%==============================================================================
A auditoria foi realizada em \today\ utilizando \texttt{curl -sL}
(seguimento de redirects) contra \url{https://doi.org/{DOI}}.
\begin{itemize}
  \item \textbf{HTTP 200:} DOI resolvido com sucesso para página do
        artigo.
  \item \textbf{HTTP 403:} DOI resolvido para página do artigo, mas
        acesso negado (paywall).
  \item \textbf{HTTP 404/202:} DOI não encontrado no sistema CrossRef.
        Causas possíveis: DOI digitado incorretamente, DOI fabricado,
        DOI ainda não registrado, ou DOI retirado.
\end{itemize}
