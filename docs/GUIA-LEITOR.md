## Bem-vindo ao Livro Imersão! 🎉

Este não é um livro comum. Você não vai apenas **ler** — você vai **viver** a experiência 
de aprender como a Inteligência Artificial está transformando a Odontologia.

---

## 🚀 Por Onde Começar

### Se você é **dentista e nunca programou**:

1. Comece pela **Parte I** (Capítulos 1-3) — nível 🟢
2. Siga a jornada da **Dra. Marina** — ela é você!
3. Não pule as práticas — o Google Colab não morde
4. Entre no **Discord** para tirar dúvidas

### Se você é **estudante de odontologia**:

1. Capítulos 1-2 para contexto histórico
2. Capítulo 3 para o básico de Python
3. Depois siga linearmente — cada capítulo constrói sobre o anterior

### Se você é **cientista da computação**:

1. Comece pela **Parte II** (Capítulos 4-7) — teoria de modelos
2. Vá para a **Parte IV** (Capítulos 12-23) — aplicações avançadas
3. Volte para a **Parte III** (Capítulos 8-11) — metodologia SDD/TDD
4. Use os notebooks Colab para experimentar

### Se você é **pesquisador**:

1. Comece pela **Parte IV-V** — o estado da arte
2. Use as **referências** com DOI para aprofundar
3. Consulte os **fichamentos** para revisão rápida
4. Os **testes TDD** são exemplos de reprodutibilidade

## 🎮 Como Funciona a Gamificação

Cada ação sua no livro vale **XP (Experiência)**:

| Ação | XP |
|------|----|
| Ler uma seção | +50 |
| Fazer uma prática | +200 |
| Passar nos testes TDD | +500 |
| Responder um desafio | +300 |
| Completar um capítulo | +1.000 |
| Completar uma parte | +5.000 |

Ao acumular XP, você **sobe de nível** e **desbloqueia badges**:

```
Nv 1  → Curioso Digital (0 XP)
Nv 2  → Aprendiz de IA (2.000 XP)
Nv 3  → Praticante Clínico (5.000 XP)
Nv 4  → Desenvolvedor Odontológico (12.000 XP)
Nv 5  → Arquiteto de Soluções (25.000 XP)
Nv 6  → Pesquisador IA-Odonto (50.000 XP)
Nv 7  → Mestre em Gêmeos Digitais (100.000 XP)
Nv 8  → PhD em Odontologia IA (200.000 XP)
```

> 💡 **Dica:** Use o script `python3 livro/imersao/gamification/xp_system.py` 
> para registrar seu progresso!
